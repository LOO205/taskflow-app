# TaskFlow

TaskFlow est une petite application de **gestion de projets collaboratifs** : tu peux te connecter avec un compte défini dans `db.json`, voir la liste des projets sur un tableau de bord, ouvrir la fiche d’un projet, en créer de nouveaux, les renommer ou les supprimer. Le front est fait avec **React**, **TypeScript** et **Vite** ; les données passent par **json-server** (fichier `db.json`) en local. Les séances de TP ont ajouté au fil de l’eau le routage, l’authentification, le CRUD, des variantes d’interface (MUI / Bootstrap) et la gestion d’état avec **Redux Toolkit** pour coller aux consignes du cours. Pour lancer l’app : dans un premier terminal `npm run api`, dans un second `npm install` puis `npm run dev` ; compte démo typique : `admin@taskflow.com` / `admin123`.
